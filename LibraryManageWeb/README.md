# LibraryManageWeb

## 实战：编写图书管理系统 (Servlet+Thymeleaf+Mybatis)

### 项目需求
+ 书管理员的登陆和退出（只有登陆之后才能进入管理页面）
+ 图书的列表浏览（包括书籍是否被借出的状态也要进行显示）以及图书的添加和删除
+ 学生的列表浏览
+ 查看所有的借阅列表，添加借阅信息

---

### 项目结构

采用三层结构：`表示层`、`业务逻辑层`、`数据访问层`
